# Rabvik-Innovations

## Tech Stack
- Frontend: HTML/CSS/Bootstrap/ReactJS
- Backend: NodeJS/MongoDB

## Quick Start

- Fork and Clone the repo using
```
git clone https://github.com/vanshika-bhasin/Rabvik-Innovations.git
```
- Install dependencies using
```
npm install
```
- Run server using
```
npm start
```
- Then in Browser

localhost:3000
